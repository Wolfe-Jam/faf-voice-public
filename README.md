## What FAF Voice is

Voice-to-FAF demo — real-time voice context persistence for AI assistants

Part of the **FAF Family** — the IANA-registered Foundational Context Layer for AI. `.faf` is the format ([`application/vnd.faf+yaml`](https://www.iana.org/assignments/media-types/application/vnd.faf+yaml)), and FAF Voice is one of the products that builds on it.
